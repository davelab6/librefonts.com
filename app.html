Hand-Picked Tales from PANCHATANTRA with Hand-Picked Type from Google Fonts

There are over 640 Google web fonts available for free. Problem is, pairing typefaces isn't easy. And, many of the fonts in Google's library don’t work well when applied to typical webpage (desktop) layouts. This collaborative, ongoing project helps provide typographic inspiration for using Google’s web fonts for web applications.
